## Quick reference

- **Stack:** Zola static site (`mode = "docs"`), Tanuki theme as a git submodule
  at `themes/tanuki` (run `git submodule update --init --recursive` before building).
- **Build:** `zola build` (Zola 0.22.1).
- **Deploy:** push/merge to `main` → GitHub Actions builds and publishes to the
  `gh-pages` branch automatically. Do NOT run `./build` manually (CI handles it).
- **Affiliate links:** always link to `/goto/<slug>/`, never raw affiliate URLs.
  Generated by `scripts/gen-goto.py`.
- **Site-specific values** (analytics, ad, tracking IDs) live under `[extra]` in
  `config.toml`; every ad/tracking feature is gated by its key.
